# Alfastack-task

## App installation:

* YOU SHOULD HAVE FRAPPE AND ERPNEXT INSTALLED FIRST

* bench get-app https://github.com/alaa-jaraa/Alfastack-task.git

* bench --site {your-site} install-app 

* bench migrate

* bench restart
